# Bytebank

Projeto em Angular (10.1.6).

## Rodar o projeto  

**Criar uma API rest de mock**  

1. Instale a biblioteca JSON SERVER `npm install -g json-server`

1. No terminal, vรก para a pasta **dados** e rode o comando `json-server --watch db.json`, para subir a API
